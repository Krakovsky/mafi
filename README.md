
# Mafia Game React Project

## Visuals & Animation
- Use Three.js for 3D city background and player models.
- Player models: Cinematic, stylized, similar in volume and look to Detroit: Become Human (placeholder assets initially).
- Day/Night cycle: Sun/moon, lighting changes.
- 10 player models in a circle, facing each other.
- Animated actions: Neutral character shoots, doctor saves, etc.

## Game Logic
- Roles: Mafia, Sheriff, Doctor, Citizens.
- Phases: Night (actions), Day (discussion/voting).
- Moderator controls for phase transitions and role management.

## Libraries
- React, Three.js, Zustand (state management), styled-components (UI), placeholder 3D assets.
